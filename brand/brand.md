# Countrify — Brand Reference

Fuente: `manual countrify.pdf` (Digital Amenities, 2025).

## Identidad

- Audiencia: residentes de countries y barrios cerrados.
- Propuesta: ecosistema de beneficios exclusivos (gastronomía, wellness, eventos, educación, actividades familiares, reservas, alertas comunitarias).
- Sin costo para el usuario final.
- Dominio: `countrify.com.ar`.
- Atribución: "Digital Amenities".

## Paleta

| Token              | Hex       | Uso sugerido            |
| ------------------ | --------- | ----------------------- |
| `brand.navy`       | `#112250` | Color principal         |
| `brand.navy-soft`  | `#3b507d` | Secundario / hovers     |
| `brand.sand`       | `#f4dcb3` | Acento cálido           |
| `brand.gray-200`   | `#d9d6d6` | Gris medio              |
| `brand.gray-100`   | `#eaeaea` | Gris claro / fondos     |

## Tipografía

- Titulares / display: **Montserrat Alternates**
- Cuerpo / UI: **Montserrat**

## Logo

- Isotipo: casa estilizada con ondas wifi.
- Logo principal: isotipo + wordmark "Countrify" sobre fondo claro.
- Logo secundario: isotipo + wordmark en blanco grisáceo sobre fondo `#112250`.
- Reducción: solo isotipo (para favicons, iconos de app, avatares).

## Tono y estilo visual

- Sofisticado pero cercano.
- Base blanco grisáceo + acentos azul.
- Bordes redondeados, mucho aire, jerarquía clara.
- Mobile-first.
